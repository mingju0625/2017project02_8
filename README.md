### 計畫摘要：
1. 利用beautifulsoup爬取PTT股票版106/12/05-107/01/05股票版資料
2. 利用jieba提取關鍵詞
3. 判斷出每日標題被討論最多次的股票，並且作為當天分析對象
4. 抓取TEJ當其股價及報酬資料，並整理成excel資料表
5. 匯入python並做資料分析
6. 結論
